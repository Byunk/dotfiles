# dotfiles

Makes setting up a new machine a breeze.

## Usage

```bash
# Install Homebrew
make install-homebrew
# Install all packages with homebrew
make install-packages
# Install all dotfile configurations
make install
# Install macOS settings
make macos-setting
```
